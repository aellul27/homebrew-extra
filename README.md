# Aellul27 Extra

## How do I install these formulae?

`brew install aellul27/extra/<formula>`

Or `brew tap aellul27/extra` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "aellul27/extra"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
